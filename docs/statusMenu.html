기본 상태창

메인메뉴 : 기본적으로 프린팅 준비, 점검, 동작에 필요한 기능들이 포함된 메뉴 모음.
상태 메세지 창 : 프린터의 상태와 에러 메세지, 경고 메세지 등을 확인할 수 있음.
출력 : 출력 파일이 포함된 저장 공간(SD, USB, Micro SD)을 확인하고 실행할 수 있음.
